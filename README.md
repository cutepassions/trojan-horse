# trojan-horse
## 프로젝트 개요
### 기간
- 2022.03 ~ 2022.04.22 (약 2개월)
### 인원
- 5인 (조용민, 김도현, 박소현, 송현준, 진병욱)
### 역할
- 조용민 (팀장, client.py)
- 김도현 (ddos.py)
- 박소현 (encryption.py)
- 송현준 (messagerelayserver.py)
- 진병욱 (keylog.py)
### 프로젝트 목적
- JAVA 업데이트를 가장한 바이러스를 분석
- 실제 공격이 이루어지는 python과 공격자가 공격을 실행시키도록 도와주는 JAVA로 구성
